Section 2: An Introduction to Web Development

  • 011.   The 'internet' is the infrastucture that connects billions of different devices all across the globe using a method called routing where individual machines are identified by their IP address, it is what makes the working of web, email, streaming, gaming possible.
  • 012.   World Wide Web is a system that allows exchange of documents over the internet using the HTTP protocol, according to which a client sends a request and receives a response from a server identified by its URL.
  • 013.   The response received from a server is in the form of HTML-CSS-JS files which are interpreted by the client's browser and then displayed.
  • 014.   server-side logic = backend, client-side code = frontend
  • 015.   HTML = noun, CSS = adjective, JS = verb

Section 3: HTML - The Essentials

  • 019.   Hypertext Markup Language is a syntax used to describe the structure of a webpage
  • 020.   Bold tag <b>
  • 021.   MDN is an invaluable resourse
  • 022.   Paragraph tag <p>
  • 023.   Headings <h1>, <h2>, <h3>, <h4>, <h5>, <h6>. A page should only have one h1 element. Use heading to indicate something important not for size
  • 025.   HTML boilerplate
    •      <!DOCTYPE> declaration informs the browser of the version of HTML being used
    •      <html> tag is the root element of the document
    •      <head> element contains machine-readable information about the document
    •      <title> shows up on the browser tab and search engines
    •      <body> contains all content of the webpage
  • 027.   <ol> stands for Ordered List and <ul> stands for Unordered List
  • 028.   Anchor tag <a> is used to create a hyperlink
  • 029.   <img> is an empty element contains, 'src' attribute for an image

Section 4: HTML - Next Steps & Semantics

  • 033.   Inline elements share space (span), Block-level element take up the whole width (div)
  • 034.   common HTML elements
    •      <hr> creates a horizontal line along the width of the elements
    •      <br> makes a line break
    •      <sup> elevates text off the baseline
    •      <sub> renders text below baseline
